### 标签选择器

+ 标签选择器的样式：
> **Html 标签名 { 属性:值;}**

+ 常用的属性：

> + color --> 前景色（文字颜色）
> + background-color --> 背景色
> + font-size --> 字体大小
> + width --> 设置宽度
> + height --> 设置高度
